# 監造計畫與品質計畫製作綱要 — 重點整理

> 來源：行政院公共工程委員會 監造計畫製作綱要 + 品質計畫製作綱要（109.04.27）
> 頁數：76 頁
> 整理日期：2026-06-15

---

## 總覽

本文件包含兩大綱要：

1. **監造計畫製作綱要**（第 1–39 頁）— 監造單位（第二層級品保）使用
2. **品質計畫製作綱要**（第 40–76 頁）— 施工廠商（第一層級品管）使用

> 兩者皆以 **五千萬元以上工程規模** 為基準編訂，較小規模可適度縮減。

---

## Part 1：監造計畫製作綱要

### 第一章 監造範圍
- **依據**：服務契約、工程契約、技師法、建築法、營造業法、職安法…等
- **工程概要**：名稱、主辦機關、設計/監造單位、地點、期限、規模、預算
- **主要施工項目及數量**：從契約詳細表中篩選重要項目（勿直接抄錄）
- **適用對象**：監造單位、廠商、材料供應商、分包商
- **名詞定義**：避免認知差異

### 第二章 監造組織及權責分工
- 組織架構圖（含管理階層、工地部門、派駐人員配置）
- 工作職掌以 **職稱** 說明，不針對特定人名
- 依 4 表權責分工（有/無委託專案管理廠商，公有/公共工程）
- 開工前召開協調會議，傳達設計理念、監造標準、契約規定

### 第三章 品質計畫審查作業程序
- 審查核定流程（含流程圖）、審查時限
- 不符合處理（補件、退回、重送）
- 對廠商品管人員之審查及核定
- **審查重點**（表3.1）：計畫範圍、管理權責、施工要領、品質管理標準、材料與設備檢驗程序、自主檢查表、不合格品管制、矯正預防措施、內部品質稽核、文件紀錄管理系統

### 第四章 施工計畫審查作業程序
- **分階段送審**：整體施工計畫 + 分項施工計畫
- **整體施工計畫審查重點**（表4.1，13項）：
  1. 工程概述
  2. 開工前置作業（地質複勘、地下管線調查…）
  3. 施工作業管理（人員組織、機具設備…）
  4. 進度管理（要徑作業、里程碑…）
  5. 假設工程計畫（工區配置、臨時水電…）
  6. 施工測量
  7. 施工區域排水系統
  8. 分項工程施工計畫
  9. 職業安全衛生管理計畫
  10. 緊急應變及防災計畫
  11. 環境保護執行計畫
  12. 施工交通維持及安全管制措施
  13. 移交管理計畫
- **分項施工計畫審查重點**（表4.2）：工項概要、人員組織、預定作業進度、分項品質計畫、職安管理、施工圖說、相關附件

### 第五章 材料與設備抽驗程序及標準
- **材料設備送審管制總表（表5.1）**：項次、材料名稱、契約數量、是否取樣試驗、預定送審日期、是否驗廠、送審資料（協力廠商資料/型錄/試驗報告/樣品）、審查結果
- **材料設備檢(試)驗管制總表（表5.2）**：預定進場日期、進場數量、抽樣日期、抽樣頻率、累積數量、抽試驗結果
- **材料/設備品質抽驗紀錄表（表5.3）**
- **抽驗標準**：抽驗項目、標準、時機、方法、頻率、不符合處理、管理紀錄
- 不合格品退料 + 矯正預防措施 + 追蹤管制

### 第六章 設備功能運轉測試抽驗程序及標準
- **三階段測試**：
  1. **單機設備測試**：進場前檢驗 → 進場組裝抽驗
  2. **系統運轉測試**：獨立功能 → 系統組合 → 清理排放
  3. **整體功能試運轉**：整合測試 → 操作紀錄 → 點交訓練
- 標準表（表6.2）：管理項目、標準、時機、方法、頻率、不合格處理

### 第七章 施工抽查程序及標準
- **抽查方式**：檢驗停留點（hold point）＋ 不定期抽查
- **檢驗停留點至少應包括**：
  1. 材料進場時
  2. 施工完成後即無法目視查看之隱蔽作業點
  3. 影響安全或結構強度之關鍵作業點
  4. 影響使用功能之關鍵作業點
  5. 工項施作完成時
- **施工抽查標準（表7.2）**：施工流程、管理項目、抽查標準、時機、方法、頻率、不符合處置、管理紀錄
- **施工抽查紀錄表（表7.3）**

### 第八章 品質稽核
- **外部稽核**：對廠商品質計畫及施工計畫執行成效
- **內部稽核**：監造單位對監造計畫落實程度
- **稽核流程**：通知 → 起始會議 → 現場稽核 → 稽核後會議 → 結果通知 → 矯正預防 → 結案

### 第九章 文件紀錄管理系統
- **文件**：公文書信、契約、圖說、計畫書、出廠證明、空白表格…
- **紀錄**：查驗紀錄、會議紀錄、日報表、自主檢查表、稽核紀錄、施工照片…
- 分類編碼、登錄收發、核定保存、作廢、移轉存檔

## Part 2：品質計畫製作綱要

### 第一章 計畫範圍
- 同監造計畫第一章，但增加 **廠商及專任工程人員** 資訊

### 第二章 管理權責及分工
- **品管組織架構圖（圖2.1）**：
  - 管理階層（含主任技師/建築師）
  - 工地主任 → 品管人員、現場工程師、行政人員
- 品管人員受工地主任管轄＋公司品管部門業務監督（虛線）
- 品管人員 **不得兼任其他職務**、**不得跨越其他標案**
- 品管人員背景應與工程性質相符

### 第三章 施工要領
- 列出需訂定之分項工程施工要領項目（表3.1，25項建築工程範例）
- 內容至少包含：施工機具、使用材料、施工方法/步驟/流程圖、注意事項

### 第四章 品質管理標準
- 各分項工程品質管理標準一覽表（表4.1）
- **品質管理標準表（表4.2）**：
  - 依施工流程（材料/施工前/施工中/施工後）區分
  - 欄位：管理項目、管理標準、檢查時機、檢查方法、檢查頻率、不合格處理、管理紀錄
- **管理標準務必量化**，訂定容許誤差；定性為輔

### 第五章 材料與設備及施工檢驗程序
- 材料/設備送審管制總表（表5.1，同監造版）
- 材料設備檢(試)驗管制總表（表5.2，同監造版）
- **材料自主檢查表（表5.3）**
- 施工檢驗流程（自主檢查 → 通知監造查驗）
- 檢驗停留點需以書面告知監造單位

### 第六章 設備功能運轉檢測程序及標準
- 機電系統架構圖
- 三階段檢測：單機 → 系統 → 整體
- 整體功能試運轉應提交：全程操作紀錄、異常檢測報告、試運轉報告、操作模式、點交訓練計畫

### 第七章 自主檢查表
- 各分項工程自主檢查表一覽表（表7.1，25項）
- **自主檢查表（表7.2）**：檢查項目、檢查標準、實際檢查值、檢查結果
- 由現場工程師或領班實地檢查後簽名，**不宜事後蓋章**
- 自主檢查表 **不應有主辦機關或監造單位簽署欄位**

### 第八章 不合格品之管制
- **材料不合格**：退料、區隔儲存、追蹤管制、與矯正預防措施連結
- **施工不合格**：即時改正 vs 重大缺失（不同管制方式）
- 不合格率異常時啟動矯正措施

### 第九章 矯正與預防措施
- **矯正**：消除已發生缺失之原因，防止再發
- **預防**：消除潛在原因，防患未然
- 品管人員應辦理統計分析，據以提出矯正預防
- 矯正措施成效不佳 → 重行檢討；成效良好 → 回饋至預防措施

### 第十章 內部品質稽核
- **稽核重點**：
  1. 人員具備基本知能與責任認知
  2. 了解工作標準
  3. 各項計畫落實執行
  4. 文件/紀錄佐證
  5. 成果符合品質要求
  6. 回饋機制有效性
- 定期 + 不定期稽核

### 第十一章 文件紀錄管理系統
- 隱蔽部分應輔以施工照片紀錄
- 文件應標示日期及版次，防止誤用失效文件

## 關鍵提醒

| 項目 | 重點 |
|------|------|
| **管理標準** | 盡量 **量化**，訂定容許誤差；定性為輔 |
| **檢查時機 ≠ 頻率** | 時機=何時檢查，頻率=多久/多少量檢查一次 |
| **檢驗停留點** | 明確標示於管理標準表或抽查時機欄位 |
| **自主檢查** | 現場人員簽名，監造單位不簽署（獨立性） |
| **監造不簽自主檢查表** | 監造抽查 ≠ 廠商品管，應使用各自表單 |
| **綱要為原則性規範** | 契約有規定時從其規定 |
| **五千萬元以上基準** | 未達者可適度縮減（如未達一千萬，分項品質計畫可併入整體） |
| **機電設備** | 運轉類設備工程須增訂「設備功能運轉檢測」章節 |
| **變更設計** | 涉及材料或工法變更時，應即時修訂分項品質計畫 |
