# 監造計劃書撰寫、審查與操作 — 重點整理

> 來源：義守大學 詹明勇（2020.05.04 高雄市政府）
> 頁數：75 頁
> 整理日期：2026-06-15

---

## 一、核心觀念

### 三級品管制度
- **第一級**：廠商施工品質管制（品質計畫）
- **第二級**：監造單位品質保證（監造計畫）
- **第三級**：主辦機關工程施工查核

### 計畫書審訂依據（由大至小）
`法規/準則 → 共同規範 → 契約 → 圖說 → 施工規範 → 特別條款`

---

## 二、依工程規模之計畫內容區分

| 工程金額 | 品質計畫（廠商） | 監造計畫（監造單位） |
|----------|-----------------|---------------------|
| **<1000萬** | 管理權責、材料檢驗程序、自主檢查表 | 監造組織、品質計畫審查、施工計畫審查、材料抽驗、施工抽查 |
| **1000萬~5000萬** | + 計畫範圍、品質管理標準、文件紀錄 | + 監造範圍、文件紀錄管理系統 |
| **>5000萬** | + 施工要領、不合格品管制、矯正預防、內部稽核 | + 品質稽核（完整版） |

### 品管人員配置
| 金額 | 要求 |
|------|------|
| <2000萬 | 品管無須專任 |
| 2000萬~2億 | 品管專任 1 人 |
| >2億 | 品管專任 2 人 |

### 監造現場人員最低人數
| 金額 | 人數 |
|------|:----:|
| 查核金額↑～未達巨額 | 至少 1 人 |
| 巨額採購 | 至少 2 人 |

## 三、各章節撰寫與操作重點

### 第一章 監造範圍
- 依據、工程概要、適用對象、名詞定義
- **主要施工項目及數量**：從契約詳細表篩選，不可直接抄錄
- 監造計畫應於**工程發包前提報甲方審核，開工前完成核定**

### 第二章 監造組織
- **人員職稱應統一**：監造工地負責人、監造主任、監造現場人員
- **不應簡稱為「品管人員」**（監造方 vs 廠商品管人員）
- 工作職掌以職稱說明，不針對特定人名
- 開工前召開協調會議，傳遞設計理念與管理標準

### 監造現場人員工作重點（17項）
1. 訂定監造計畫，監督查證廠商履約
2. 審查施工計畫、品質計畫、預定進度、施工圖、器材樣品
3. 審查重要分包商及設備製造商資格
4. 訂定檢驗停留點（限止點），會同廠商取樣送驗
5. 校驗放樣、施工基準測量
6. **抽查施工作業及抽驗材料設備，填具抽查(驗)紀錄表**
7. 發現缺失通知廠商限期改善，確認改善成果
8. 督導工地安全衛生、交通維持、環境保護
9. 審核履約進度及估驗計價
10. 協調及整合履約界面
11. 契約變更建議及協辦
12. 監督機電設備測試及試運轉
13. 審查竣工圖表、結算明細表
14. 驗收協辦
15. 協辦履約爭議處理
16. 填報監造報表
17. 其他工程事宜

### 第三章 品質計畫審查作業程序
- **審查流程**：含流程圖、審查時限（收件→審查→審定→補件→退件→核定）
- **品管人員審查**：資格、人數、更換作業規定
- **版次管制**：送審情形管制表
- **審查重點**：管理責任、施工要領、品質管理標準、材料檢驗程序、自主檢查表、不合格品管制、矯正預防、內部稽核、文件紀錄
- **錯誤樣態**：
  - 未訂審查時限 → 提送或審查期程過久
  - 未訂補件時限
  - 未訂定計畫完成時限
  - 品質/施工計畫未核定前先行施工
  - 未考量工程規模予以簡化

### 第四章 施工計畫審查作業程序
- 整體施工計畫（13項，與綱要相同）
- 分項施工計畫審查：工項概要、人員組織、預定進度、分項品質計畫、職安管理

### 第五章 材料與設備抽驗程序及標準
- **材料設備送審管制總表**（監造管控）
- **材料檢試驗管制總表**（承商主控）
- 材料設備應依規定送審合格，始得進場及施工
- 需控管之材料設備依據：**詳細價目表、資源統計表**
- **出廠證明判讀案例**：
  - 圖說要求 SD280W（可銲），出廠證明為 SD280 → 不合格
  - 圖說要求 SUS316，出廠證明為 SUS304 → 不合格
- **材料進場時查驗重點**：核對是否與送審合格者相符

### 第六章 設備功能運轉測試
- 三階段：單機測試 → 系統運轉測試 → 整體功能試運轉
- 系統運轉測試項目：獨立功能、系統組合、清理排放、應用表單
- 整體試運轉應提交：全程操作紀錄、異常檢測報告、試運轉報告、操作模式、點交訓練計畫

### 第七章 施工抽查程序及標準
- **施工抽查時機**：檢驗停留點（hold point）+ 不定期抽查
- **檢驗停留點**：開工前明確告知廠商，顯示於管理標準表
- **監造單位不應在廠商自主檢查表上簽證**
- **施工品質抽查紀錄表內容**：抽查項目、抽查標準、實際抽查情形、抽查結果
- **缺失改善追蹤三級處理**：
  1. 可立即完成改善 → 抽查表缺失複查 + 改善前中後照片
  2. 未能立即完成改善 → 缺失改善追蹤表 + 照片
  3. 施工缺失頻率高及重大缺失 → 不合格報告 + 矯正預防措施

### 施工抽查標準常見錯誤
| 錯誤類型 | 說明 |
|----------|------|
| 管理項目欠具體 | 管理標準無法精確訂定 |
| 管理標準未量化 | 未訂定容許誤差 |
| 檢查時機與頻率混淆 | 何時檢查 vs 多長時間/數量檢查一次 |
| 不符合處理不切實際 | 文字說明過於含糊 |
| 管理紀錄文件不清 | 未清楚訂定須留存之合格證明文件 |
| 抽查標準未對應主要工項 | 應配合主要工項訂定抽查標準 |
| 管理紀錄誤植 | 應為抽查紀錄表，非自主檢查表 |
| 抽查情形未確實填寫 | 照抄抽查標準、未填量測值 |
| 抽查人員未簽名 | 應由監造現場人員填寫並立即簽名 |
| 抽查標準與規範不符 | 如鋼筋保護層未依施工規範選用允差 |

### 具體錯誤案例
- **鋼筋規格錯誤**：現場用 SD280W，圖說需 SD420W
- **PVC管厚度錯誤**：現場用 A 管（薄），圖說需 B 管（厚）
- **電纜線錯誤**：現場用 IV 電線，圖說需電纜線
- **抽查結果判讀錯誤**：實測高程差 5.9cm > 允許差 3cm，卻判合格
- **CLSM 抽查標準錯誤**：將混凝土坍度標準誤用於 CLSM
- **模板支撐間距不符**：計算書 80cm，抽查標準卻訂 120cm
- **保護層厚度錯誤**：標準 2cm±6mm，實測 5cm 卻判合格

### 第八章 品質稽核
- **內外稽核不同**，應分別書寫執行要領
- **外部稽核**：對施工廠商
- **內部稽核**：監造單位自我稽核
- **查核金額以上工程**必須執行內外稽，以**二~三個月一次**為原則，宜在同一時段辦理
- **第一次稽核**：開工後 60 日內實施，重點在人員認知與組織權責
- **稽核流程**：成立小組 → 擬定計畫 → 核准 → 通知 → 前會議 → 現場作業 → 後會議 → 撰寫報告 → 缺失改善 → 結案

### 第九章 文件紀錄管理系統
- **文件分類**：與廠商往返書函、監造計畫、施工計畫、施工圖說、品質計畫、檢試驗計畫、抽驗紀錄表、查核紀錄表、往來公文
- **管理程序**：登錄 → 收發 → 核定（含送業主）→ 保存 → 作廢
- **編碼原則**：文件碼 + 流水號（如 AB12-001）
- **工程完工後**：移轉予業主、規劃存檔位置及年限

## 四、實戰下手順序（第73-74頁）

```
接到設計圖說 → 限期完成初版
  ├─ 檢閱設計圖與契約書
  ├─ 契約書內工程項目與材料總表
  ├─ 施工規範
  ├─ 監造計畫製作綱要
  ├─ 經驗傳承 / 範本 / 範例
  │
  1. 第一、二章（監造範圍、組織）
  2. 計畫書審查（第三、四章）
  3. 材料檢驗與流程（兩張總表）
  4. 施工抽查（停檢點、檢驗標準、查驗表）
  5. 書面文件系統（缺失、稽核、編號）
  6. 機水電試運轉
```

## 五、關鍵提醒

| 項目 | 重點 |
|------|------|
| 監造 vs 品管 | 監造現場人員不應簡稱「品管人員」，兩者角色不同 |
| 自主檢查表 | 監造單位不簽署（獨立性原則） |
| 抽查紀錄表 | 由監造現場人員**親自填寫**並**立即簽名** |
| 出廠證明判讀 | 務必比對圖說/契約規格，不可只看型式名稱 |
| 材料進場查驗 | 核對是否與送審合格者相符，不可直接放行 |
| 抽查標準量化 | 管理標準應量化 ± 容許誤差 |
| 施工流程導向 | 抽查標準應依**完整施工流程**訂定，非依工種切割 |
| 不合格三級處理 | 即時改正 / 追蹤改善 / 矯正預防（依嚴重程度分級） |
| 檢驗停留點 | 開工前明確告知廠商，顯示於標準表 |
