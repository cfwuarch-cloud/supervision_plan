# 材料設備品質抽驗紀錄表轉換 — 計畫說明

## 目的

將詳細價目表（Excel）作為母本，每項材料產出一份空白抽驗紀錄表（表5.3），供現場填寫抽驗結果。

## 輸入

| 檔案 | 說明 |
|------|------|
| `data/02_成德-詳細價目表.xlsx` | 詳細價目表母本（Table 1 工作表） |
| `tables/table5.3/表5.3.docx` | 表5.3 模板 |

## 處理規則

### 資料篩選
- 起點：項次 `壹.三.1` 之後
- 項次須以 `壹.` 開頭
- 排除單位：`式`、`工`（可自訂 `--exclude-units`）
- 排除名稱含 `小計`、`合計`、`總價` 之項目

### 產出
- 每項材料產出一份空白表單（所有欄位保留模板原始內容）
- 每份表單之間插入分頁符號

### 模板結構
- 表頭 4 列：文件編號、工程名稱、材料設備名稱/使用位置/供料廠商、廠牌型號/契約數量/檢查日期
- 抽驗項目區（6 欄）：檢查項目、抽驗標準、抽驗紀錄、符合、不符合、複驗結果
- 結尾區：抽驗意見、監造人員簽名

## 使用方式

```bash
python -X utf8 tables/table5.3/convert_5.3.py --exclude-units 式 工
```
