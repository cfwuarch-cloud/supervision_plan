# 不合格改善追蹤表轉換 — 計畫說明

## 目的

將詳細價目表（Excel）作為母本，每項材料產出一份不合格改善追蹤表（表5.5）。

## 輸入

| 檔案 | 說明 |
|------|------|
| `data/02_成德-詳細價目表.xlsx` | 詳細價目表母本（Table 1 工作表） |
| `tables/table5.5/表5.5.docx` | 表5.5 模板 |

## 處理規則

### 資料篩選
- 起點：項次 `壹.三.1` 之後
- 項次須以 `壹.` 開頭
- 排除單位：`式`、`工`（可自訂 `--exclude-units`）
- 排除名稱含 `小計`、`合計`、`總價` 之項目

### 產出
- 每項材料產出一份追蹤表
- 填入：工程名稱、監造單位、材料名稱
- 其餘欄位保留空白（不符情形、原因調查、改善情形、改善成果確認）

## 使用方式

```bash
python -X utf8 tables/table5.5/convert_5.5.py --exclude-units 式 工
```
