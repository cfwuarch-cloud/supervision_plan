# Ch1 監造範圍轉換 — 計畫說明

## 目的

產出監造計畫第一章「監造範圍」docx 文件。

## 輸入

| 檔案 | 說明 |
|------|------|
| `data/02_成德-詳細價目表.xlsx` | 詳細價目表母本 |
| `tables/ch1/ch1_內容.md` | 章節文字內容（待提供） |

## 處理規則

### 資料篩選（價目表）
- 沿用 table5.1 篩選邏輯：起點壹.三.1、排除單位 式/工
- 新增 `--level` 參數控制階層深度

### 文字內容
- 讀取 markdown 章節：依據、工程概要、適用對象、名詞定義
- `{{專案名稱}}` 等 placeholder 由命令列參數替換

### 輸出結構
1. 第一章 監造範圍
   - 1.1 依據
   - 1.2 工程概要
   - 1.3 工程主要施工項目及數量（價目表表格）
   - 1.4 適用對象
   - 1.5 名詞定義

## 使用方式

```bash
python -X utf8 tables/ch1/convert_1.py --level 4
python -X utf8 tables/ch1/convert_1.py --level 4 --project-name "自訂專案名稱"
```
