# 圖7.1 輕隔間施工抽查流程圖 — 計畫說明

## 目的

產製輕隔間施工抽查流程圖（圖7.1）Word 文件。

## 輸入

| 檔案 | 說明 |
|------|------|
| `data/圖7.1_輕隔間施工抽查流程圖.docx` | 原始流程圖（VML 格式） |

## 處理方式

- 從原始檔提取 VML 流程圖元件（矩形流程框、箭頭連線、菱形判斷框、註解框等）
- 以 python-docx 建構新文件，保留相同段落結構與頁面設定
- 將 VML group 注入 P2（流程圖容器段）
- VML 包含 13 個流程框、8 條連線、背景形狀及文字方塊

## 輸出

| 產出 | 說明 |
|------|------|
| `output/圖7.1_輕隔間施工抽查流程圖.docx` | 輕隔間施工抽查流程圖 |

## 使用方式

```bash
python -X utf8 tables/table7.1/convert_7.1.py
```
