# 抽驗結果通知單轉換 — 計畫說明

## 目的

將詳細價目表（Excel）作為母本，每項材料產出一份抽驗結果通知單（表5.4）。

## 輸入

| 檔案 | 說明 |
|------|------|
| `data/02_成德-詳細價目表.xlsx` | 詳細價目表母本（Table 1 工作表） |
| `tables/table5.4/表5.4.docx` | 表5.4 模板 |

## 處理規則

### 資料篩選
- 起點：項次 `壹.三.1` 之後
- 項次須以 `壹.` 開頭
- 排除單位：`式`、`工`（可自訂 `--exclude-units`）
- 排除名稱含 `小計`、`合計`、`總價` 之項目

### 產出
- 每項材料產出一份通知單
- 填入：工程名稱、結構物或材料名稱
- 其餘欄位保留空白（契約規範標準、抽驗結果、處理情形、備註）

## 使用方式

```bash
python -X utf8 tables/table5.4/convert_5.4.py --exclude-units 式 工
```
