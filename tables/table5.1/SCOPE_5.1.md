# 材料送審管制總表轉換 — 計畫說明

## 目的

將詳細價目表（Excel）作為母本，填入材料送審管制總表（表5.1）Word 模板，產出完整多頁 Word 文件。

## 輸入

| 檔案 | 說明 |
|------|------|
| `data/02_成德-詳細價目表.xlsx` | 詳細價目表母本（Table 1 工作表） |
| `tables/table5.1/表5.1.docx` | 表5.1 模板 |

## 處理規則

### 資料篩選
- 起點：項次 `壹.三.1` 之後
- 項次須以 `壹.` 開頭
- 排除單位：`式`、`工`（可自訂）
- 排除名稱含 `小計`、`合計`、`總價` 之項目
- ⑤ → 窗（修正 PDF 轉檔錯誤）

### 分頁
- 動態分頁，依資料列實際高度決定每頁可容納組數
- 每頁之間插入分頁符號

### 標題列合併（R1 + R2）
- **合併**：C0（項次）、C2（契約數量）、C3（是否取樣試驗）、C6（預定試驗單位）、C14（備註）
- **不併**：C1、C4、C5、C7~C12、C13

### 資料列合併（奇數列 + 偶數列）
- **合併**：C0（流水號）、C2（數量+單位）、C3、C6、C7~C12、C14
- **不併**：C1（奇=項次、偶=名稱）、C4、C5、C13

## 使用方式

```bash
python -X utf8 tables/table5.1/convert_5.1.py --exclude-units 式 工
```
