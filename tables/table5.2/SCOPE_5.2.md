# 材料設備檢(試)驗管制總表轉換 — 計畫說明

## 目的

將詳細價目表（Excel）作為母本，填入材料設備檢(試)驗管制總表（表5.2）Word 模板，產出完整多頁 Word 文件。

## 輸入

| 檔案 | 說明 |
|------|------|
| `data/02_成德-詳細價目表.xlsx` | 詳細價目表母本（Table 1 工作表） |
| `tables/table5.2/表5.2.docx` | 表5.2 模板 |

## 處理規則

### 資料篩選
- 起點：項次 `壹.三.1` 之後
- 項次須以 `壹.` 開頭
- 排除單位：`工`（可自訂 `--exclude-units`）
- 排除名稱含 `小計`、`合計`、`總價` 之項目
- 文字清洗：移除換行符、多餘空格、標點前空格
- ⑤ → 窗（修正 PDF 轉檔錯誤）

### 分頁
- 動態分頁，預先計算每頁可容納組數
- 每頁可用高度：`DATA_AVAIL_TWIP`（10600）− `HEADER_H_TWIP`（1100）≈ 9500 twip
- 每頁約 9 組資料（視文字長短浮動）
- 每頁之間插入分頁符號

### 表頭（每頁頂部 3 行）
1. `表5-2 材料設備檢(試)驗管制總表-N`（14pt、粗體、置中、固定行高 24pt）
2. `工程名稱：...`（12pt、靠左）
3. `(監造單位使用) 第N頁共Σ頁 表單編號：E52-N`（12pt、最小行高 12pt）

### 儲存格格式
- 黑實線框（1pt、黑色、`exact`）
- 段落行高：`exact` 240 twip（與列高一致，防文字裁切）
- 垂直置中

## v2.0 重大變更

| 項目 | 舊版 | v2.0 |
|------|------|------|

| 行高校準 | 無 | 段落 `exact` 行高 + `LINE_H_TWIP` |

| 頁碼表頭 | 無 | 三行表頭（標題、工程名稱、頁碼） |

## 輸出

| 產出 | 說明 |
|------|------|
| `output/表5.2_完成.docx` | 完整多頁材料設備檢(試)驗管制總表 |

## 使用方式

```bash

python -X utf8 tables/table5.2/convert_5.2.py --test-num 1 --exclude-units 工 式

# 正式輸出
python -X utf8 tables/table5.2/convert_5.2.py -o output/表5.2_完成.docx
```
