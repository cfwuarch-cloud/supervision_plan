# 材料設備檢(試)驗管制總表轉換 — 計畫說明

## 目的

將 **詳細價目表（Excel）** 作為母本，填入 **材料設備檢(試)驗管制總表（表5.2）Word 模板**，產出完整的多頁 Word 文件。

## 輸入

| 檔案 | 說明 |
|------|------|
| `02_成德-詳細價目表.xlsx` | 詳細價目表母本（Table 1 工作表） |
| `表5.2.docx` | 表5.2 模板（1 個表格、2 列標題 + 2 組空白資料列） |

## 處理規則

### 資料篩選
- 起點：項次 `壹.三.1` 之後
- 項次須以 `壹.` 開頭
- 排除單位：`工`（可自訂）
- 排除名稱含 `小計`、`合計`、`總價` 之項目
- ⑤ → 窗（修正 PDF 轉檔錯誤）

### 分頁
- 動態分頁，依各資料列實際高度（Pillow 測量字寬 + 行高校準）決定每頁可容納組數
- 每頁總高度上限：11500 twip（標題列 971 + 資料列 10529）
- 每頁之間插入分頁符號

### 標題列
- 10 欄位：項次、契約詳細表項次／材料(設備)名稱、預定進場日期／實際進場日期、進場數量、抽樣日期／抽樣數量、規定抽樣頻率、累積進場數量／累積抽樣數量、檢(試)驗結果、檢(試)驗及會同人員、備註／(歸檔編號)

### 標題列合併（R0 + R1）
- **合併**：C0（項次）、C3（進場數量）、C5（規定抽樣頻率）、C7（檢(試)驗結果）、C8（檢(試)驗及會同人員）
- **不併**：C1（項次/名稱）、C2（進場日期）、C4（抽樣）、C6（累積數量）、C9（備註/歸檔編號）

### 資料列合併（奇數列 + 偶數列）
- **合併**：C0（流水號）、C3、C5、C7、C8
- **不併**：C1（奇=項次編碼、偶=材料名稱）、C2、C4、C6、C9

## 輸出

| 產出 | 說明 |
|------|------|
| `表5.2_完成.docx` | 完整多頁材料設備檢(試)驗管制總表 |

## 使用方式

```bash
# 基本用法（使用預設路徑）
python -X utf8 supervision_plan/supervision_plan/table5.2_converter/convert_5.2.py

# 指定檔案
python -X utf8 supervision_plan/supervision_plan/table5.2_converter/convert_5.2.py \
    -p 02_成德-詳細價目表.xlsx \
    -t 表5.2.docx \
    -o output.docx

# 自訂每頁組數與排除單位
python -X utf8 supervision_plan/supervision_plan/table5.2_converter/convert_5.2.py --max-pairs 15 --exclude-units 工
```
